# Product Vision

## Overview
SERVMORX TECH is a field-first HVAC diagnostic system.

Goal:
Help technicians diagnose faster without relying on phone calls.

---

## Value proposition
"Ditch the phone calls."

---

## Product wedge
Start with:
- diagnostic tool

Expand later to:
- CRM
- equipment history
- analytics

---

## Equipment intelligence (NEW)
The app should support:
- data tag scanning
- automatic model detection
- API-based spec retrieval
- warranty lookup
- common failure patterns

---

## Long-term moat
- real field logic
- structured routing
- equipment-aware reasoning
- dataset of service outcomes
